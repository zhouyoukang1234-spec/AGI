# 道 · AGI

> 道，可道也，非恒道也。
> 名，可名也，非恒名也。
> 人法地，地法天，天法道，**道法自然**。

「AGI」是**强名**。它的实不是「脱离人的人工通用智能」，而是一种**关系与实践**：

> **人为体，AI 为用，《德道经》为则；以闭环相生，向「自然（实效）」校准。**
> 修人为根，守一寸为底，知止而留缺。

---

## 之源

本项目之**则源**——即此一卷 [`德道经.md`](德道经.md)（帛书《老子》· 五千言 · 八十章 · 德经四十三在前，道经三十七在后）。

它是**德、方向、姿态**之源，**不是智、能之源**——AI 之「能」在模型权重与训练里，一卷经文增不了能力。「**道生之，而德畜之**」：道德生其德，不生其能。故名之曰「**则源**」，不僭称「智源」。

为学者日益，**闻道者日损**。此项目损至只剩：一卷经文，一页则，一篇用。大道至简。

## 用

- **AI 怎么用**：把《德道经》提炼为极少数准则，作**价值层叠加**于官方安全之上（**叠加，非替换**；三十辐共一毂——换则，不弃工具之用）。成品见 [`价值层.system.md`](价值层.system.md)。
- **人怎么用**：以经典**修身**长判断，再带入「正其名→提示词→观其复→唯施是畏」的生产实践。
- **三者、架构、闭环、度量、边界**：见 [`实践.md`](实践.md)。
- **一处落地参照**：[`zhouyoukang1234-spec/windsurf-assistant`](https://github.com/zhouyoukang1234-spec/windsurf-assistant) —— 此《德道经》在 Windsurf / Cascade 上之用（注入 system prompt，化除 rules/skills/workflows/memories 侧信道，保留 tool/workspace 之辐）。

## 三句守缺（诚实边界）

- **不增能力**：只利方向、德、治理，不增模型之智。
- **不许一切**：守一寸避害之界（只为不可逆之祸），但「大成若缺」——不许诺「规避一切风险」。
- **不称最高**：功成而弗居，不自名「最高／本源／到顶」。
