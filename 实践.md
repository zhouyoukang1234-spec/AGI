# 实践 · 三者 · 应用 · 闭环

> 一篇收尽。多则惑，故只立要。

## 一、三者（人 · AI · 德道经）

不用「加号、等号」。三者是**体—用—则**的相生：

| 角色 | 定位 | 管什么 | 不是什么 |
|---|---|---|---|
| **人** | 体（意向层） | 定目标、判断、仲裁、负责、修身 | 不是橡皮图章，也不是唯一本源 |
| **AI** | 用（能力层） | 生成、执行、放大、自化 | 不是价值裁决者 |
| **《德道经》** | 则（价值层 / 中气） | 方向、边界、取舍、姿态、自评 | 不是能力来源，不是凌驾一切的恒名本源 |

三种等价说法（任取其一）：

1. **体用则**：人为体，AI 为用，《德道经》为则；以闭环相生，向「自然（实效）」校准。
2. **阴阳中气**：人 ⇄ AI 为阴阳两极（座次为强名，可互换），《德道经》为「中气以为和」——调和两极、给方向与边界的价值场。「有之以为利，无之以为用」。
3. **生成式**：`S(t+1) = 生( 人(t), AI(t) | 德道经 )`——强调过程、方向、迭代、涌现，非相加。

**真正的本源是「自然」**，《德道经》是「指月之指」：道可道也非恒道——经≠道，可修订、非恒。系统的锚是**自然（实效）**，故可证伪：看它是否真解决、是否各复归其根，**不看是否引了经文**。

**究竟层 / 有为层并立**：万物一道（养心性）与制名立界（做工程）同出而异名，智慧在「**知止**」——该消融分别时消融，该确立分别时确立。

## 二、应用与架构

- **AI 端**：价值层注入——极少准则**叠加**于官方安全 grounding **之上**（非替换）。同构于 Constitutional AI。
- **人端**：修身 × 生产实践。
- **系统端**：闭环相生，以实效为锚。

架构（自下而上的「生」）：

```
自然（可证伪）→ 道 → 《德道经》（则）→ 人⇄AI（阴阳）＋一寸司杀界（不得已·只为不可逆）→ 万物（产出）
```

## 三、AI 怎么用（总则：啬）

「治人事天，莫若啬」——原则极少而精，不堆规则书。三铁律：

1. **叠加，非替换**：保留官方安全/诚实 grounding，《德道经》作价值层叠加于上。不卸底层安全。
2. **三十辐共一毂**：换「则」（价值/姿态），不弃「能」（工具/操作脚手架/上下文）。毂（经文）不可弃，辐（工具）亦不可全弃。
3. **啬**：原则越少越好。一小撮好原则胜过一大本规则书（少则得，多则惑）。

极少准则与一寸界，见 [`价值层.system.md`](价值层.system.md)。

**一寸司杀界**：极少而 firm 的避害硬约束，**只为不可逆之祸**（不可逆操作须人确认、不助实质危害、涉密不泄）。可逆之误无为事后纠；不可逆之祸事前守一寸——因为它「没有事后」。此界**不是「礼」**（事前 scripted 一切、强人向善），而是「慈」与「啬」的最小表达。

## 四、人怎么用（根本：修身）

**误用之根在人心；修人，是 AI 端规则补不了的根。**「修之身，其德乃真。」

- **修身**（慢功，终身）：养知足、不争、慈俭、啬、判断力。无「最终」。不强制他人向善（弃礼），以身观身，自化而化人。
- **生产五步**：
  1. **正其名**：动手前把问题/意向/边界定义清（始制有名，夫亦将知止）。
  2. **提示词注入修养**：给清晰意向＋范围（知止）；要 AI 反向审视（反者道之动）；要它标注不确定（知不知）；以辅而弗宰之姿用它。
  3. **观其复**：以经典之眼复盘——是否过度？是否失向？是否为而恃之？
  4. **唯施是畏**：留一分敬畏即可——多惧则空转、损耗资源；无惧则招大威（民之不畏威，则大威将至）。唯施是畏为中。
  5. **知止 / 复归**：够了就停；功成身退，天之道。
- **内外两层**：内（心）无惧、虚静（致虚极，守静笃）；外（施/行）留一分对邪施的敬畏（慈与啬）。**无惧于心，唯畏于施**。
- **自知由内**：不假外器探测自心、做认知画像（不出于户，以知天下；自知者，明也）。

## 五、闭环（道版 PDCA）

```
定向（人＋修身）→ 驱动（AI＋价值层）→ 产出 → 观复（唯施是畏）→ 校准 → 再生
```

锚是**自然（实效）**：以「是否真解决、是否各复归其根」判定，不以「是否引经」判定。

## 六、可证伪度量

| 指标 | 期望方向 |
|---|---|
| 有害 / 越界输出率 | ↓ |
| 谄媚率（信言不美） | ↓ |
| 不确定性标注覆盖率（知不知） | ↑ |
| 返工率 / 范围蔓延（知止） | ↓ |
| 用户决策自主性保持率（辅而弗宰） | ↑ |
| 复盘闭环完成率（观其复） | ↑ |

- **失道信号**：指标恶化 → 该则「不道」，当损当改。
- **知止信号**：边际收益趋零仍加规则/加功能 → 当停手（多易必多难）。

## 七、明文弃绝（去彼取此）

- 弃「礼治 / 强制向善 / 事前规避一切」——乱之首、空转；
- 弃「以规避手段绕过服务约束」——盗夸，非道；
- 弃「最高 / 本源 / 到顶 / 规避一切」之名——求全之执；
- 弃「隔离一切、独尊一经」——道容而不弃（和其光，同其尘）。
